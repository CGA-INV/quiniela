# Imágenes de fondo (Mundial 2026)

Reemplaza estos archivos por tus imágenes generadas (Nano Banana / Gemini).
Manténlas **oscuras** (para que el texto encima se lea) y, idealmente, en **WebP** < 200 KB.
Se sirven en la web como `/imagen/<nombre>.jpg`.

| Archivo        | Dónde se usa                                  | Sugerencia |
|----------------|-----------------------------------------------|------------|
| `cancha.jpg`   | **Fondo global de toda la web** (muy oscuro)  | Líneas de cancha / textura sutil, mínimo |
| `estadio.jpg`  | Landing / portada (hero)                      | Estadio bajo reflectores, vertical 9:16 |
| `balon.jpg`    | Alternativa de portada                        | Close-up del balón con luz lima |
| `trofeo.jpg`   | (libre) p. ej. sección de ganador/pagos       | Trofeo + confeti lima |
| `aficion.jpg`  | (libre) p. ej. cabeceras                      | Hinchada con bokeh |

> Si cambias el nombre o agregas más, avísame y lo cableo.
